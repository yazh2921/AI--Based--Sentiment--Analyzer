# AI Based Sentiment Analyzer

## Description

This project analyzes user reviews and predicts whether the sentiment is Positive, Negative, or Neutral using Python and TextBlob.

## Technologies Used

* Python
* TextBlob
* Google Colab

## Features

* Positive sentiment detection
* Negative sentiment detection
* Neutral sentiment detection

## Example Reviews

Positive:
"This product is amazing"

Negative:
"I hate this product"

Neutral:
"This is a product"
